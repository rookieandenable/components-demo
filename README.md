## demo component library


### 引用

~~~javascript
npm install enable-demo --save
~~~

### 使用

~~~javascript
// 加载样式
import 'enable-demo/dist/index.css'
// 引入组件
import { Button } from 'enable-demo'
~~~

### 组件

* 🔥Button
* 🔥Input

### 推荐node版本
~~~bash
node version: 14.x
~~~
### 本地启动命令

~~~bash
//启动本地环境
npm run storybook

//单元测试
npm run test

//打包构建静态文件
npm run build-storybook

//发布静态文件
npm run build

//发布到 npm
npm publish
~~~
